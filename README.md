# Transportation_Logistic_Analysis_Project
Analyzes shipment performance, delivery efficiency, and route patterns using Power BI. Transforms raw logistics data into interactive dashboards with actionable KPIs for operational optimization and performance monitoring.

🎯 Project Goals
Track on-time vs delayed deliveries (670 shipments analyzed)

Analyze transportation distance patterns (Avg: 183.92 KM)

Identify high-volume shipment locations

Optimize logistics operations through data insights

📊 Dataset
  2019-2020 Logistics Data:-
  • Booking ID & Shipment Dates
  • Origin/Destination Locations
  • Vehicle Types
  • Delivery Status
  • Transportation Distance
  • GPS Provider Performance

🔧 Data Pipeline:-
  1. Data Cleaning (Missing values, formatting)
  2. Calculated Measures Creation
  3. Power BI Dashboard Development
  4. Interactive KPI Visualization

📈 Dashboard Sections:-
  Executive Summary
  KPIs Tracked:
  • Total Shipments: 670
  • On-Time: 204 (30%)
  • Delayed: 469 (70%)
  • Avg Distance: 183.92 KM

Performance Analysis
  • Vehicle type performance
  • Origin location trends
  • GPS provider comparison
  • Delivery status breakdown

Route Optimization
  • Distance distribution analysis
  • Route efficiency metrics
  • Vehicle-distance correlation
